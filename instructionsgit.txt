1. делаем, чтобы была возможность пользоваться гитом
git init
2. Содать ветку и перейти в нее
git checkout -b newpage
3. Добавить файлы для коммита
git add . (в этом случае добавляем все)
git add название файла (это добавляем выборочное)
4. Коммитем добавленное
git commit -m "added new title"
5. Для отправки данных в гитхаб (интернет)
git push -u origin newpage
6. Выгрузка с интернета производится
git clone https://github.com/DmitryNV/super.site имя папки куда нужно выгрузить
7. Посмотреть, где мы находимся (желтая строчка)
8. Посмотреть несохраненные изменение
git status
9.cd - перейти в папку
10. dir - показать, что находится в папке
11. cd .. - вернуться в папку родителя (выйти из настоящей папки)
